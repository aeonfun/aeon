## aaronjmars/aeon

> **Highlights:** Every Aeon fork now ships a public `/status/` health dashboard, extended this morning with a Token Pulse row (price, 24h, liquidity, volume, FDV). Five new skills landed alongside it — `contributor-reward`, `skill-analytics`, `fork-skill-digest`, an `./onboard` validator, and the AdManage trio (`aixbt-pulse`, `schedule-ads`, `create-campaign`).

### Added
- Public `/status/` page now renders a Token Pulse row (price, 24h, liquidity, 24h volume, FDV) above the skill-health table on every heartbeat run. ([#146](https://github.com/aaronjmars/aeon/pull/146))
- `SHOWCASE.md` lists the most active forks and compares Aeon against AutoGen, CrewAI, n8n, and LangGraph, with a one-line pointer added to the README. ([#145](https://github.com/aaronjmars/aeon/pull/145))
- New `contributor-reward` skill turns the weekly fork-contributor leaderboard into a tier-priced USDC rewards plan that `distribute-tokens` can execute. ([#144](https://github.com/aaronjmars/aeon/pull/144))
- New `skill-analytics` skill produces a Wednesday fleet-wide ranking of skill runs and flags silent, all-failing, or low-success skills. ([#142](https://github.com/aaronjmars/aeon/pull/142))
- New public `/status/` page gives every fork a live health dashboard with overall verdict, per-skill table, and open issues, auto-regenerated by heartbeat three times daily. ([#141](https://github.com/aaronjmars/aeon/pull/141))
- New `fork-skill-digest` weekly skill flags where the configured fork fleet has converged on different `enabled`, `var`, `model`, or `schedule` defaults than upstream ships. ([#140](https://github.com/aaronjmars/aeon/pull/140))
- New `./onboard` CLI and matching skill validate eight setup checks on a fresh fork and print one-line fix commands for any gaps. ([#139](https://github.com/aaronjmars/aeon/pull/139))
- New `aixbt-pulse`, `schedule-ads`, and `create-campaign` skills add a cross-domain market pulse and AdManage.ai-backed ad provisioning, paused-by-default with daily spend caps and dry-run guards. ([#138](https://github.com/aaronjmars/aeon/pull/138))
